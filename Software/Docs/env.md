# CW32 环境搭建教程

## 开源工具链搭建

CW32 开源构建工具链主要分为以下几部分：

1. arm-none-eabi-gcc
2. GNU make
3. pyOCD
4. Visual Studio Code

### arm-none-eabi-gcc

在 ARM 官网下载 arm-none-eabi-gcc 后安装

注意：**安装时勾选添加到环境变量，否则需要手动添加**

### GNU make

在 GNU 官网或者 MSYS2 中安装 make for windows 并添加至环境变量

### pyOCD

在 Python 官网安装 python

在 CMD 或者 Powershell 中执行

``` cmd
pip install pyocd
```

### Visual Studio Code

在 VS Code 官网下载安装包安装
