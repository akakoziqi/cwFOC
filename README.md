# cwFOC

![MCU](https://img.shields.io/badge/MCU-CW32F030-brightgreen) ![License](https://img.shields.io/badge/License-GPL--v2-blue)

本项目是一个基于芯源半导体 CW32F030K8 的小尺寸 FOC 驱动器

![渲染图](./assets/cwFOC_3d.jpg)

## 控制器进度

- [x] 点灯
- [x] AS5600 驱动
- [x] MP6540 驱动
- [x] SVPWM 开环
- [x] SVPWM 位置闭环
- [ ] 电流采样
- [ ] SVPWM 电流闭环
- [ ] 速度闭环控制
- [ ] 位置闭环控制
- [x] 定点计算优化
- [ ] 通信协议
